## 🟣 W&B

| 항목      | TensorBoard | W\&B                               |
| ------- | ----------- | ---------------------------------- |
| 설치 및 연동 | 로컬 서버 필요    | 클라우드 기반, 바로 공유 가능                  |
| 실험 추적   | 로그 저장만 가능   | 실험 비교, config 기록, resume까지         |
| 시각화     | 제한적         | 예쁜 대시보드, 다양한 뷰 (Loss, 이미지, Grad 등) |
| 협업      | X (로컬 전용)   | O (링크 공유, 팀 협업)                    |
| 커스터마이징  | 제한됨         | 코드 + GUI + Sweep 가능                |
| 이미지 로그  | 가능하지만 불편함   | `.log_image()`로 간편하게 지원            |
